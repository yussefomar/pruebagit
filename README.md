# Proyecto de Práctica Git

Proyecto sencillo para practicar **HTML, Git, ramas, Pull Requests y conflictos de merge**.

## Contenido

La página contiene:

- Un título `Hola mundo`.
- Una lista de elementos.
- Un título `<h2>` al final que dice `Termina el día`.

## Tecnologías utilizadas

- HTML
- Git
- GitHub

## Objetivo

Practicar el flujo de trabajo con Git y GitHub:

1. Crear ramas.
2. Realizar cambios.
3. Hacer commits.
4. Subir ramas a GitHub.
5. Crear Pull Requests.
6. Realizar merge a `main`.
7. Resolver conflictos de merge.
